### 🚀 Projets Sélectionnés

#### ![Hardware](https://img.shields.io/badge/-Hardware-239120?style=flat-square) **[Hardware Monitor](https://github.com/Ryn-GH/HardwareMonitor)**
Développement d'une application de monitoring système en **C# / WPF**.
* Utilisation de **LibreHardwareMonitor** pour l'accès aux capteurs Ring 0.
* Implémentation d'un pattern **MVVM** et gestion du multithreading asynchrone pour la fluidité de l'UI.

#### ![Web](https://img.shields.io/badge/-Web-FF2D20?style=flat-square) **Maritime Dashboard**
Conception d'une plateforme d'analyse maritime avec **API REST Laravel**.
* Système de cache optimisé (Hit/Miss) pour les données environnementales.
* Visualisations cartographiques interactives avec **Leaflet** et **Chart.js**.
